# Mini LinkedIn

Build a Mini LinkedIn-like Community Platform

## Getting Started

```batch
npm install
npm run dev
```


## Features

### User Authentication

- Register/Login (Email & Password)
- Profile with name, email, bio

### Public Post Feed

- Create, read, display text-only posts
- Home feed with author’s name and timestamp

### Profile Page

- View a user’s profile and their posts

## Tech Stack

- Frontend: Next.js
- Backend: Firebase
